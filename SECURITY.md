# Security

> The full security policy — supported versions, vulnerability reporting,
> response SLAs, severity classification, and disclosure rules — lives at
> [`.github/SECURITY.md`](.github/SECURITY.md), the location GitHub auto-detects
> for the **Security** tab. This file exists only to keep the compliance-doc
> index discoverable from the repository root.

**Report vulnerabilities** via [GitHub Security Advisories](https://github.com/sebastienrousseau/bankstatementparser/security/advisories/new). Do not open a public issue.
